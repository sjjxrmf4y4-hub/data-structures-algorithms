# Student Class in C++

Object-oriented C++ project implementing a Student data model with validation and formatted output.

## Features

- Header and implementation file separation
- Default and parameterized constructors
- Getter and setter methods
- GPA validation
- Date validation including leap years
- Phone number validation
- Major validation
- Formatted console output

## Skills Demonstrated

- C++ classes
- Encapsulation
- Input validation
- String handling
- Modular code organization
- Build tools with Makefile

## Files

- student.h
- student.cpp
- makefile

## Outcome

Strengthened understanding of object-oriented programming and reusable class design in C++.
