# Linked List in C++

## Node Structure

```cpp
struct Node {
    int data;
    Node* next;
};


## Core Operations

- Insert at front
- Insert at end
- Delete node
- Traverse list
- Search for value

## Advantages

- Dynamic size
- Efficient insertions/deletions
- No contiguous memory required

## Tradeoffs

- No direct indexing
- Extra memory for pointers

## Learning Outcome

Linked lists help explain dynamic memory and pointer-based structures.
